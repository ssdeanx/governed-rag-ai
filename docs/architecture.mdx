# System Architecture

The Mastra Governed RAG is built around Mastra's multi-agent orchestration framework, integrating Next.js for the frontend/API, Qdrant for vector storage, and OpenAI for embeddings/LLM. The core logic resides in `src/mastra/`, with workflows coordinating agents, tools, and services for secure RAG operations.
